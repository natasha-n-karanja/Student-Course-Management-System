# Student Course Management System

Python capstone that uses:
- Object-Oriented Programming
- Linked Lists
- SQLAlchemy ORM
- Unit Testing

## Setup
1. Clone repo
2. `pip install -r requirements.txt`
3. Run `main.py`

## Features
- Register students/courses
- Track history with Linked List
- Save to SQLite DB
- Run unit tests with `unittest`
